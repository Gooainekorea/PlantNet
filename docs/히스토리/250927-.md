여전히 학습에 너무 오래걸린다.

- HDD에 저장했더니 GPU랑 CPU의 속도를 못따라와서. SSD인 C 드라이브로 옮겼다.
하지만 여전히 30%이하로 사용되고 어제 돌려놓고 나갔는데 오늘까지 끝나지 않았다. CPU사용률은 50%이상이다.

- 모델, 데이터를 .to(device)로 GPU로 올리라는데 이미 그렇게 하고 있다.

- 이미지 처리 파이프라인에서 병목현상 발생한 것 같다.
    지나가시던 교수님이 작은 파일 수만장인 구조에 따라 다수 load 하는 방식은 I/O레벨에서 병목 발생 한다고 한다
