# 자주쓴 코드

1. 가상환경 활성화
.\가상환경디렉토리명\Scripts\Activate.ps1
.\.venv\Scripts\Activate.ps1

2. 설치된 torch, torchvision 삭제
pip uninstall torch torchvision torchaudio

3. 재설치
pip install torch==2.8.0+cu129 torchvision==0.23.0+cu129 --index-url https://download.pytorch.org/whl/cu129


4. 버전 확인
Python 버전 확인: python --version
PyTorch 버전 확인: import torch; print(torch.__version__)
torchvision 버전 확인: import torchvision; print(torchvision.__version__)
CUDA 버전 확인: nvcc --version 또는 nvidia-smi
